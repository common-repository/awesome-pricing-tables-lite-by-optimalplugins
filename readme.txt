== Installation ==

1. Upload the Pricing Tables for WP plugin file (`awesome-pricing-tables-lite-by-optimalplugins.zip`) to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In your sidebar, select 'Pricing Tables -> Add New' to create a new table
